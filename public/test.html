Image Upload Test for TipTap Editor

This tests that the image upload functionality has been properly integrated into the TipTap editor:

Features Added:

  • ✅ Added @tiptap/extension-image dependency
  • ✅ Imported Image extension in TiptapEditor.tsx
  • ✅ Added Image to the extensions array
  • ✅ Added uploadImage function to github.ts
  • ✅ Added image upload button (📷) to the toolbar
  • ✅ Added hidden file input for image selection
  • ✅ Added handleImageUpload function
  • ✅ Integrated with editor.setImage() command

How It Works:

  1. User clicks the 📷 button in the editor toolbar
  2. File picker opens for image selection
  3. Selected image is converted to data URL via uploadImage()
  4. Image is inserted into the editor using editor.setImage()
  5. When saved, the image will be converted to markdown format

Code Changes Made:

// Added image upload functionality
const handleImageUpload = async (event) => {
  const file = event.target.files?.[0];
  if (file && editor) {
    const imageUrl = await uploadImage(file);
    editor.chain().focus().setImage({ src: imageUrl, alt: file.name }).run();
  }
};
